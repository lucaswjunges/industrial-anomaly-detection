Project Overview

Atlantic Water Operations Ltd. operates a critical water treatment facility serving 250,000 residents. The facility experiences 15-20 hours of unplanned pump downtime monthly, costing approximately $180,000 in failures, repairs, and service penalties.

This project implements an end-to-end machine learning pipeline for real-time anomaly detection in multivariate IoT sensor streams from industrial pumping equipment. The system detects equipment failures 10-30 minutes before critical breakdown, enabling proactive intervention and preventing costly downtime.

Business Impact: The deployed system prevents an estimated 38 hours of unplanned downtime during the 6-day evaluation period, yielding a 30-day net value of $3.1 million through early failure detection.

Technical Approach

Sensor Architecture

Each industrial pump is monitored via six sensors sampled at 1-minute intervals:

  • Temperature (RTD Pt100): Detects friction, electrical issues
  • Vibration (Accelerometer): Identifies bearing wear, cavitation
  • Pressure (Piezoelectric): Monitors hydraulic performance
  • Flow Rate (Electromagnetic): Tracks throughput efficiency
  • Current (Hall Effect): Detects electrical faults, load changes
  • Duty Cycle (VFD): Operational state indicator

Engineering Highlights

1. Regime-Aware Preprocessing

Conventional normalization causes false alarms during operational state transitions (startup, high-load, maintenance). Our regime-specific normalization reduces false positives by 74% by accounting for expected operational variations.

2. Physics-Informed Feature Engineering

Beyond raw sensor values, we extract domain-specific features:

  • Rolling statistics: 5-minute mean/std for trend detection
  • Temporal derivatives: Rate-of-change indicators
  • Cross-sensor ratios: Pressure/flow, temp/vibration (captures interaction effects)

Result: +18% F1-score improvement over raw features alone.

3. Autoencoder Architecture

Deep neural network trained to reconstruct normal operational patterns. Anomalies exhibit high reconstruction error due to deviation from learned baseline.

Architecture: [27 → 16 → 12 → 8 → 12 → 16 → 27] with batch normalization and dropout for regularization.

4. Deployment Flexibility

Supports three deployment architectures based on operational requirements:

  • Edge: NVIDIA Jetson Nano ($99) for low-latency detection (<100ms)
  • Cloud: AWS SageMaker for centralized multi-facility analytics
  • Hybrid: Edge (Isolation Forest) + Cloud (Autoencoder) for best-of-both
